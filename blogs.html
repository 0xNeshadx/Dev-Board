Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in the DOM, which are Id, Class name, Tag name, Name attribute, Css selector, and Traversing the Dom.

Here are various methods available for selecting elements:
By Id using getElementById(), by class name using getElementsByClassName(), by tag name using getElementsByTagName(), by name attribute using getElementsByName(), by css selector using querySelector() and querySelectorAll(), and lastly traversing the DOM using properties like children, firstElementChild, nextElementSibling.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

For innerHTML sets the HTML content inside an element, including tags. Also, it is possible to modify the HTML structure dynamically.
For innerText sets only the visible text inside an element. Also, ignore hidden elements and extra whitespace.
For textContent sets the full text inside an element, even if it's hidden.

Question-4: What is event bubbling in the DOM?

Event bubbling is a term in the DOM where the event an element receives is bubbled transmitted to its parent and upward in the DOM tree, until it gets to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

These methods allow to create, add, and remove elements using in Javascript:

To create a new element for use, document.createElement("tagName")
To add the element to the parent for use, parentElement.appendChild(newElement)
To remove an element for use, parentElement.removeChild(elementToRemove) or element.remove()
